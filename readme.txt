Bitte Notebooks immer über den "Run All" Knopf ausführen.

Zum Beginnen ensamble.ipynb ausführen. Dort sind alle Modelle enthalten, diese werden alle ausgeführt und alle einzelnen Ergebnisse werden gespeichert. Diese Ergebnisse werden dann zu einem Gesamtergebnis zusammengeführt, indem der Durchschnitt genommen wird. 

Die Ergebnisse der einzelnen Modelle lassen sich in modell_1.csv bis modell_3.csv einsehen. Das Gesamtergebnis in submission.csv.

Auf Kaggle haben wir modell2_new submittet und einen Score von 0.640 bekommen: https://www.kaggle.com/code/ericaidanjacobsen/top-25-parameter-model

Weitere Informationen finden Sie ganz oben in jedem Notebook.

Inhalt:

ensamble.ipynb: Alle Modelle und Gesamt-Prediction
explore_data.ipynb: Explorative Datenanalyse
feature_importance.ipynb: Eine Auflistung der Feature Importance
modell1.ipynb: Modell 1 (RandomSurvivalForest)
modell2_new.ipynb: Modell 2 (XGBRegressor mit Hyperparametertuning und top Feature Importance)
modell2_old: Ältere Version von Modell 2 (Wird nicht in Ensemble genutzt)
modell3.ipynb: Modell 3 (LGBMClassifier)
story_telling.ipynb: 2 Reißerische Grafiken
top_features_model.ipynb: EIne Weitere ältere Version von Modell 2 (Wird nicht in Ensemble genutzt)
